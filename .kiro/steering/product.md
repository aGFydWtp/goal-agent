# Product Overview

半期の評価目標を Discord Bot 経由で継続追跡する個人向け Agent。ユーザーは週次で「今週やったこと」を雑に入力するだけでよく、Agent が内容を各評価目標に分類・証跡化し、進捗判定・不足指摘・次アクション提案・自己評価文ドラフトまでを Discord チャット UI 上で完結させる。専用 Web ダッシュボードは持たない。

## Core Capabilities

- **週次チェックイン**: 雑な自然文入力を評価目標へ自動分類し、確認のうえ証跡化する(製品の中心)
- **証跡管理**: 活動を「評価可能な証跡」として蓄積し、目標との関連度を保持
- **進捗判定**: 各目標を Green/Yellow/Red/Gray で見立て、不足と次アクションを提示
- **評価文ドラフト生成**: 蓄積証跡から事実/解釈/課題/次アクションを分離した自己評価文を生成
- **定期通知**: 週次チェックイン催促と状態悪化アラートを能動的に送る

## Target Use Cases

- 半期評価目標を複数持ち、週次で振り返りたい個人
- 評価前に実績整理の負担を減らしたい人

初期スコープ外: チーム全体の目標管理、上司による監視、人事評価システム連携。

## Value Proposition

期初に立てた目標が日々の業務で意識されず流れてしまう問題を、「毎週聞いてくれるチャット Agent」で解決する。検証する中心仮説は **「評価目標管理はダッシュボードより、毎週聞いてくれるチャット Agent の方が継続する」**。成功はリリース機能数ではなく継続利用(4週連続チェックイン、各目標に証跡3件以上)で測る。

## Privacy Posture

評価目標・自己評価文は個人情報性が高い。DM/個人用非公開チャンネル限定、他ユーザーデータ非アクセス、保存前のユーザー確認必須、自動分類は即確定しない、生成評価文は必ずドラフト扱い、削除コマンドを用意——これらは機能ではなく**製品の不可欠な前提**である。
